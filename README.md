This is a simple Bank Management System made using Object Oriented Programming (OOP) concepts in C++.  
The project allows basic banking operations like adding accounts, depositing money, withdrawing money and saving account details to a file.

## Features
- Create and store bank account details
- Search account by account number
- Deposit amount into an account
- Withdraw amount (with balance check)
- Save account records to text file
- Uses Exception Handling for error cases
- Uses Template and Friend Function concepts
